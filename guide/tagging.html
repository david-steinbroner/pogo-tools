Using Tags in Pokemon GO

Tags are a built-in feature in Pokemon GO that let you organize your collection. After running Collection Triage, use tags to keep things sorted.

Step 1: Create Your Tags

We recommend these tags to match our triage results:

  • PvP - Pokemon to keep for battles
  • Raid - Pokemon to keep for raids
  • Trade - Pokemon safe to trade away

Optional extra tags if you want more detail:

  • GL - Great League specific
  • UL - Ultra League specific
  • ML - Master League specific
  • Lucky Trade - High priority for lucky trades
  • Power Up - Pokemon you want to invest in next

How to create a tag:

  1. Open Pokemon GO
  2. Go to your Pokemon storage
  3. Tap the tag icon in the top right (looks like a label)
  4. Tap "Create New Tag"
  5. Type the name and pick a color
  6. Tap "OK"

Step 2: Download Your Checklist

After running Collection Triage, download the Action Checklist. It lists exactly which Pokemon to tag, with their Name and CP so you can find them.

Example checklist:

=== TAG AS "PVP" (3) ===
[ ] Azumarill CP 1487 (0/14/15)
    Rank #1 Great League - excellent!
[ ] Swampert CP 1496 (0/15/13)
    Rank #2 Great League - excellent!
[ ] Galarian Stunfisk CP 1491 (0/13/15)
    Rank #6 Great League - excellent!

=== TAG AS "RAID" (2) ===
[ ] Machamp CP 2847 (15/14/13)
    Top normal, rock, steel raid attacker
[ ] Tyranitar CP 3456 (15/12/14)
    Top fire, flying, ice raid attacker

=== TAG AS "TRADE" (2) ===
[ ] Machop CP 542 [Shadow]
[ ] Eevee CP 648

=== SAFE TO TRANSFER (3) ===
3 Pokemon can be transferred for candy.
...

Step 3: Apply Tags

Work through your checklist:

  1. Open Pokemon GO
  2. Search for the Pokemon (e.g., type "Azumarill" in search)
  3. Find the one matching the CP on your checklist
  4. Tap to open it
  5. Tap the tag icon (bottom right of the Pokemon screen)
  6. Select the appropriate tag
  7. Check it off your list and move to the next one
Pro tip: You can tag multiple Pokemon at once! Long-press one Pokemon, then tap others to select them, then tap the tag icon.

Faster method with search:

If you have many of the same Pokemon (e.g., multiple Machamp), use search with CP to find the exact one:

  1. Type: machamp&cp1234 (replace 1234 with the actual CP)
  2. This will show only Machamp with that exact CP
  3. If there are still multiple, check IVs to find the right one

Step 4: Use Your Tags

Now you can filter by tag anytime:

  • Before trading: Filter by "Trade" tag to see all your trade candidates in one place.
  • Building a team: Filter by "PvP" to see your best battle Pokemon.
  • Before a raid: Filter by "Raid" to quickly pick your attackers.
  • Cleaning up: Filter by excluding all your tags to find untagged Pokemon you might want to transfer.

How to filter by tag:

  1. Go to your Pokemon storage
  2. Tap the tag icon
  3. Tap the tag you want to filter by
  4. Only Pokemon with that tag will show

Or type in search: tag:PvP or tag:Raid

Step 5: Mass Transfer

After tagging your keepers, you can safely mass-transfer everything else:

  1. In Pokemon storage, type: !tag:PvP&!tag:Raid&!tag:Trade
  2. This shows Pokemon with NONE of your tags
  3. Review the list to make sure nothing important is there
  4. Long-press to select multiple
  5. Tap "Transfer" to send them to the Professor
Be careful! Double-check before mass transferring. Favorites, shinies, and legendaries have extra protection, but it's still good to review.

Useful Search Strings

Pokemon GO has a powerful search feature. Here are useful searches to combine with tags:

Search What it does
tag:PvP Show Pokemon tagged "PvP"
!tag:PvP&!tag:Raid&!tag:Trade Show Pokemon with NONE of your tags (unprocessed)
tag:PvP&!evolve PvP Pokemon that are fully evolved
4* Show 100% IV Pokemon (4-star)
3* Show 82-98% IV Pokemon (3-star)
0*,1*,2* Show Pokemon with lower IVs
shiny Show all shinies
lucky Show all lucky Pokemon
shadow Show all shadow Pokemon
purified Show all purified Pokemon
@special Show Pokemon with legacy/special moves
@weather Show Pokemon boosted by current weather
evolve Show Pokemon you can evolve now
item Show Pokemon that need items to evolve
costume Show Pokemon wearing costumes
legendary Show legendary Pokemon
mythical Show mythical Pokemon
cp1400-1500 Show Pokemon in a CP range
pokemon&cp1234 Find exact Pokemon by name and CP
Combine searches with &: Use & to combine multiple filters. Example: shadow&3*&fighting shows 3-star shadow Fighting-types.
